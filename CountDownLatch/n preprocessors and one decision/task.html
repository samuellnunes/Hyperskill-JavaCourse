You have n preprocessor threads. Each of these will preprocess your data. For simplicity, assume that each thread will add the prefix PRE to the data. After preprocessing, you must run a calculation thread, which will add the word Message to the data. The input is n, and the output is the final value of the data.


Sample Input:
Hello!
3

Sample Output:
Hello!PREPREPREMessage



Memory limit: 256 MB
Time limit: 5 seconds
