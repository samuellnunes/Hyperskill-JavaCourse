You are tasked with implementing a "Worker" class in a multithreaded environment. The program will spawn a specified number of threads. Each thread, upon execution, will multiply a shared counter by 2 and then terminate. The shared counter is initially set to 1.

The input to the program is the count of threads to be created. The output should be the final value of the counter.


Sample Input:
8

Sample Output:
256



Memory limit: 256 MB
Time limit: 5 seconds
