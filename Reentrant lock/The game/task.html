You are in the process of developing a game that consists of two heroes and one enemy. The enemy has a "health" value. The heroes can attack the enemy but are required to wait for 10 milliseconds after each attack. Instead of putting the thread to sleep, you wish to implement active waiting. This behavior can be achieved using the tryLock() method.

Your task is to create the Enemy class. This class should accept the current "gain" value, obtained after completing various missions, as its input. The class should then output the "gain" value following the successful execution of all attacks.


Sample Input:
80 10 10

Sample Output:
60



Memory limit: 256 MB
Time limit: 5 seconds
